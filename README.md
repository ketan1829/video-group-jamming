# Video-Group-Jamming

Videe-Group-Jamming is a simple video jam application for multi-users based on React, Node Express and WebRTC.

This project is **deployed** on https://jam.choira.io

**Technologies Used**

- React
- Node + Express
- WebRTC
- Socket.io
- [Syled-components](https://styled-components.com/)
- [Simple-peer](https://github.com/feross/simple-peer)

**Contributors:** [Hyunse Kim](https://github.com/Hyunse)

---

## Features

- Join a room
- Video Streaming
- Text chat
- Mute Video/Audio
- Screen Sharing
- Metronome

## Installation
### Clone
- Clone this repo to your local machine using

### Setup
**Client**
> Move to client folder, update and install this package
<pre>
  <code>
    /* Install */
    npm install
    
    /* Run */
    npm start
  </code>
</pre>

**Server**
> Move to server folder, update and install this package
<pre>
  <code>
    /* Install */
    npm install
    
    /* Run */
    npm run dev
  </code>
</pre>
